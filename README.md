# 📖 About The Project

AI Blog Generator is a modern web application that uses **Google Gemini AI** to generate professional, SEO-optimized blog articles from a single topic or title.

The application features a futuristic **2050-inspired user interface**, a Python Flask backend, and an interactive chat-style experience for users.

This project demonstrates modern web development skills by combining AI APIs, backend development, and responsive frontend design.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/asad816191-cell/AI-Blog-Generator.git
```

Go inside the project

```bash
cd AI-Blog-Generator
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Backend

```bash
python backend/main.py
```

Open Browser

```text
http://127.0.0.1:5000
```
